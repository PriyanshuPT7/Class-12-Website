Unit 2 – C Programming

1. Working with a Function

a. Define Function

A function is a block of code that performs a specific task. It allows code reusability and modularity in programming.

b. Syntax of a Function

return_type function_name(parameters) {
    // code to execute
}

c. Types of Functions

  • Library Functions: Built-in functions like printf(), scanf()
  • User-defined Functions: Created by programmers for custom tasks

d. Components of Function

i. Function Prototype: Declaration of the function before main()

int add(int, int);

ii. Function Call: Invoking the function to execute

add(5, 10);

iii. Function Definition: Actual implementation of function

int add(int a, int b) {
    return a + b;
}

iv. Return Type: The type of value returned (e.g. int, void)

2. Categories of Functions with Example

i. Function with return type but no arguments

int getValue() {
    int x = 10;
    return x;
}

ii. Function with return type with arguments

int add(int a, int b) {
    return a + b;
}

iii. Function with no return type and no arguments

void greet() {
    printf("Hello, World!");
}

iv. Function with no return type with arguments

void display(int n) {
    printf("Number: %d", n);
}

3. Storage Classes

i. Automatic (auto)

void func() {
    auto int a = 10;
}

ii. External (extern)

extern int x;

iii. Register (register)

register int speed = 100;

iv. Static (static)

static int count = 0;

4. Recursive Function with Syntax and Examples

A recursive function is a function that calls itself to solve a problem in parts.

int factorial(int n) {
    if(n == 0) return 1;
    else return n * factorial(n - 1);
}

Syntax of Recursive Function

return_type function_name(parameters) {
    if(condition)
        return base_case;
    else
        return function_name(updated_parameters);
}

5. Demonstration of Passing Array to a Function

void printArray(int arr[], int size) {
    for(int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
}

int main() {
    int nums[] = {1, 2, 3, 4, 5};
    printArray(nums, 5);
    return 0;
}
